# Unity Projectile Physics Demo

**Unity:** 6.4.4f1

Демонстрационный проект с собственной системой симуляции снарядов без использования встроенной физики Unity.

## Основные возможности

- Ручная симуляция движения снарядов.
- Полностью отказ от `Rigidbody` и физической симуляции Unity.
- Двухуровневая система обнаружения столкновений:
  - быстрый `SphereCast` для грубой проверки;
  - точный `RayCast` внутри сферы для получения корректной точки попадания и UV-координат.
- Процедурная генерация `Mesh` для снарядов.
- Пул объектов для снарядов.
- Пул объектов для VFX-эффектов.
- Эффекты взрывов реализованы с помощью **VFX Graph**.
- Следы от пуль на поверхностях реализованы через **Custom Render Texture**.
- Shader Graph для записи отпечатков пуль в `Custom Render Texture`.
- Отдельный Shader Graph для отображения накопленных следов на 3D-моделях.
- Эффекты камеры (отдача при выстреле и покачивание при ходьбе) реализованы с использованием **DOTween**.

## Используемые технологии
- Unity 6.4.4f1
- Shader Graph
- Custom Render Texture
- VFX Graph
- DOTween
- Unitask
